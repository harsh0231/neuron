Self-Driving and ROS 2 Course
Module 1: Introduction to ROS 2 and Self-Driving Robots
  • Overview of ROS 2: Understanding the Robot Operating System (ROS) 2 framework
  • Setting Up the Development Environment: Installing and configuring ROS 2
  • Introduction to Self-Driving Robots: Basics of autonomous navigation and self-driving technology
Module 2: Building a Self-Driving Robot
  • Hardware Components:
    • Selecting and assembling sensors, actuators, and microcontrollers
  • Robot Modeling:
    • Creating a digital twin of the robot using URDF (Unified Robot Description Format)
  • Simulation with Gazebo: Simulating the robot in a virtual environment for testing
Module 3: Odometry and Localization
  • Understanding Odometry: Calculating the robot's position and orientation over time
  • Implementing Odometry in ROS 2: Coding odometry algorithms in Python and C++
  • Localization Techniques: Applying sensor fusion and Kalman Filters for accurate positioning
Module 4: Control Systems for Self-Driving Robots
  • Introduction to Control Theory: Basics of PID (Proportional-Integral-Derivative) controllers
  • Implementing ros2_control: Utilizing the ros2_control library for hardware interface and control
  • Developing Custom Controllers: Creating and tuning controllers for specific robot behaviors
Module 5: Sensor Integration and Fusion
  • Working with Sensors: Interfacing LIDAR, IMU, and cameras with ROS 2
  • Sensor Fusion Algorithms: Combining data from multiple sensors using Extended Kalman Filters
Module 6: Advanced Topics
  • Probability Theory in Robotics: Applying probabilistic methods for decision-making
  • Path Planning and Navigation: Introduction to algorithms for autonomous path planning
  • Real-World Application: Deploying the self-driving robot in real-world scenarios
