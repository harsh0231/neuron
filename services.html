Class 5th Course Pattern

Course Duration: 1 Year

Target Group: CBSE and ICSE students (Class 5th)

Focus: Building foundational knowledge in technology, programming, IoT, and logical thinking with interactive projects.

Core Subjects Covered:

  • Basics of Computers and Technology
    • Understanding Hardware and Software
    • Input/Output Devices
    • Internet and Safe Browsing Practices
  • Introduction to Programming
    • Basics of Coding using Scratch Programming
    • Block-based logic building
  • Introduction to IoT
    • What is IoT?
    • Understanding Sensors, Actuators, and Connectivity
    • Practical Demonstrations (Using kits like Arduino for beginners)
  • Introduction to Artificial Intelligence
    • Basics of AI and its Applications
    • Fun AI Activities: Image recognition, voice commands, etc.
  • Project Management Basics
    • Planning small tasks and documenting progress
    • Teamwork and collaboration activities

Detailed Monthly Breakdown

Month 1-2: Foundational Knowledge

  • Week 1-4:
    • Introduction to technology and computers.
    • Basic Scratch Programming: Creating animations.
    • Understanding IoT and simple sensors.
    • Project 1: Smart Night Lamp using LDR sensor.
  • Week 5-8:
    • Exploring the concept of variables and loops in Scratch.
    • IoT Components: Breadboard, LEDs, and simple circuits.
    • Project 2: Automatic Plant Watering System using soil moisture sensor.

Month 3-4: Building Logical Thinking

  • Week 9-12:
    • Building small games in Scratch (Maze Game, Number Guess Game).
    • Introduction to AI through image recognition apps like "Quick, Draw!" by Google.
    • Project 3: Smart Doorbell System (PIR sensor for detecting motion).
  • Week 13-16:
    • Exploring Scratch extensions for IoT-based simulations.
    • Introduction to Python Programming (Basic Syntax, Printing, Variables).
    • Project 4: IoT-Based Weather Monitoring System (Temperature and humidity sensor).

Month 5-6: Advanced Problem Solving

  • Week 17-20:
    • Python Programming: Conditional Statements and Loops.
    • AI Activity: Creating a chatbot using simple Scratch and Python concepts.
    • Project 5: Obstacle Avoiding Car using ultrasonic sensors.
  • Week 21-24:
    • IoT: Exploring cloud platforms like ThingSpeak for storing IoT data.
    • Python Programming: Reading and writing data.
    • Project 6: Home Automation System (Controlling appliances via smartphone).

Skills Acquired by the End of Class 5:

  • Basic understanding of technology and programming.
  • Ability to design small IoT projects.
  • Logical and creative problem-solving.
  • Teamwork and project documentation skills.
  • Familiarity with AI concepts like image recognition and chatbot creation.

Class 6th Course Pattern

Course Duration: 1 Year

Target Group: CBSE and ICSE students (Class 6th)

Focus: Strengthening foundational programming, introducing IoT concepts with simple applications, and exploring beginner-friendly AI and data science topics.

Core Subjects Covered:

  • Python Programming Basics:
    • Understanding syntax, variables, and loops.
    • Simple projects with text-based outputs.
  • Introduction to IoT:
    • Hands-on work with sensors and actuators.
    • Basic Arduino projects with step-by-step guidance.
  • AI and Data Science:
    • Basics of AI concepts through fun examples.
    • Exploring patterns in small datasets (e.g., weather, scores).
  • Collaborative Project Work:
    • Team-based learning and small-group projects.

Detailed Monthly Breakdown

Month 1-2: Fundamentals

  • Week 1-4:
    • Python Basics: Variables, data types, simple input/output.
    • IoT: Introduction to sensors and actuators, basic Arduino setup.
    • Fun activities: Writing Python programs for simple math games.
    • Project 1: LED Blinking Patterns (Using Arduino).
  • Week 5-8:
    • Python Programming: Loops (for and while) and basic conditionals.
    • IoT: Using temperature and light sensors.
    • Introduction to simple data visualization (drawing bar charts on paper based on IoT data).
    • Project 2: Automatic Night Lamp (Using LDR sensor).

Month 3-4: Intermediate Learning

  • Week 9-12:
    • Python Programming: Functions (introduction).
    • IoT: Reading sensor data and displaying it on the serial monitor.
    • Basic AI Activity: Understanding how machines "see" patterns (e.g., shapes and colors).
    • Project 3: Temperature and Humidity Monitor (Using DHT11 sensor).
  • Week 13-16:
    • Python Programming: Lists and basic operations (e.g., finding the average of numbers).
    • Data Science: Collecting small datasets (like daily temperatures).
    • AI Activity: Exploring a simple chatbot concept using if-else conditions.
    • Project 4: Soil Moisture-Based Plant Watering System.

Month 5-6: Fun with AI and IoT

  • Week 17-20:
    • Python Programming: Drawing shapes using the turtle module.
    • IoT: Using ESP32 for wireless connections (pre-coded examples).
    • AI Activity: Voice commands with pre-built tools (e.g., Google Assistant).
    • Project 5: Smart Fan Control System (Using temperature sensors).
  • Week 21-24:
    • Python Advanced: Understanding how to save data to a text file.
    • Collaborative Project: Combining IoT with a small Python program.
    • Project 6: Home Automation System (Controlling lights and fans with a basic app).

Skills Acquired by the End of Class 6:

  • Basic Python programming skills (variables, loops, functions).
  • Ability to work with IoT devices like Arduino and ESP32 for simple projects.
  • Familiarity with AI concepts like chatbots and pattern recognition.
  • Understanding of how to visualize data and use it for small projects.
  • Confidence in teamwork and project management.
